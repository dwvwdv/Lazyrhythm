# lazyrhythm web
## 網站簡述
lazyrhythm開發遊戲相關作品宣傳

## 開發環境
Angular v17 + Typesript + SCSS

## 功能需求
1. 有創意的交互動作，例如推骨牌動畫
2. 需進行響應式設計，且網站要存在暗黑模式
3. 網站頁面會存放在`src/app/page`資料夾下，需要的頁面資訊如下
 - 關於公司
 - 作品集
 - 聯絡方式
 - 贊助我們
4. 共用元件會存放在`src/app/widget`資料夾下，需要的元件資訊如下
 - 導航列，會處在每個頁面的頂部
 - 頁尾
 - 圖片輪播，會出現在作品集頁面，用於展示遊戲宣傳圖
 - 表單填寫，會出現在贊助我們、聯絡我們頁面，用於收集贊助資訊
